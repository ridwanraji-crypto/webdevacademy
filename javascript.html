JavaScript Fundamentals

Variables and Data Types

JavaScript variables are containers for storing data values. Use let, const, or var to declare variables.

JavaScript Code
// Declaring variables let name = "Alice"; const age = 30; var isStudent = false; // Data types let score = 95; // Number let colors = ["red", "green", "blue"]; // Array let person = { // Object firstName: "John", lastName: "Doe" };

String Methods

Common String Methods

JavaScript provides many useful methods for working with strings, such as .length, .toUpperCase(), .slice(), and .replace().

JavaScript Code
let str = "WebDev Academy"; console.log(str.length); // 15 console.log(str.toUpperCase()); // "WEBDEV ACADEMY" console.log(str.slice(0, 6)); // "WebDev" console.log(str.replace("Academy", "School")); // "WebDev School"

Math and Date Objects

Use the Math object for calculations and Date for working with dates and times.

JavaScript Code

JSON & Local Storage

Storing Data

Use JSON.stringify() and JSON.parse() to work with localStorage.

JavaScript Code
let user = { name: "Alice", age: 25 }; localStorage.setItem("user", JSON.stringify(user)); let saved = JSON.parse(localStorage.getItem("user")); console.log(saved.name); // "Alice"

Classes & OOP

Object-Oriented Programming

Use class to create objects and inheritance in JavaScript.

JavaScript Code
class Animal { constructor(name) { this.name = name; } speak() { return this.name + " makes a noise."; } } let dog = new Animal("Dog"); console.log(dog.speak());

Modules

Import & Export

JavaScript modules help organize code. Use export and import (in supported environments).

JavaScript Code
// module.js export function greet(name) { return `Hello, ${name}!`; } // main.js import { greet } from './module.js'; console.log(greet("Alice"));

Note

Modules require a server or build tool to run in the browser.

Regular Expressions

Pattern Matching

Use RegExp to search and match patterns in strings.

JavaScript Code
let text = "My email is test@example.com"; let emailPattern = /\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z]{2,}\b/i; console.log(emailPattern.test(text)); // true

Testing JavaScript

Writing Tests

Testing helps ensure your code works as expected. You can write simple tests using functions or use frameworks like Jest.

JavaScript Code
function add(a, b) { return a + b; } function testAdd() { if (add(2, 3) === 5) { console.log("Test passed!"); } else { console.log("Test failed!"); } } testAdd();

DOM Manipulation

Selecting and Modifying Elements

The Document Object Model (DOM) is a programming interface for HTML and XML documents.

JavaScript Code
// Select elements const header = document.querySelector('header'); const buttons = document.querySelectorAll('button'); // Modify elements header.style.backgroundColor = '#323330'; header.style.padding = '20px'; // Create new elements const newElement = document.createElement('div'); newElement.textContent = 'New element created!'; newElement.classList.add('new-div'); document.body.appendChild(newElement); // Event handling buttons.forEach(button => { button.addEventListener('click', () => { button.textContent = 'Clicked!'; }); });
