CSS Fundamentals

Introduction to CSS

Cascading Style Sheets (CSS) is used to style and layout web pages. It controls the visual presentation of HTML elements, including colors, layout, fonts, and more.

CSS Code
/* Change text color and background */ body { color: #333; background: #f8f9fa; font-family: 'Arial', sans-serif; } /* Style headings */ h1 { color: #6a11cb; font-size: 2.5rem; margin-bottom: 1rem; } /* Style buttons */ .button { background: #6a11cb; color: white; padding: 0.8rem 1.5rem; border-radius: 4px; text-decoration: none; display: inline-block; }

CSS Functions

calc(), var(), clamp()

Functions like calc(), var(), and clamp() allow for dynamic and flexible values.

CSS Code

Responsive Design

Media Queries

Media queries apply CSS rules based on device characteristics like screen width.

CSS Code
/* Mobile styles (default) */ .container { padding: 1rem; } /* Tablet styles */ @media (min-width: 768px) { .container { padding: 2rem; max-width: 720px; } } /* Desktop styles */ @media (min-width: 992px) { .container { max-width: 960px; } }
