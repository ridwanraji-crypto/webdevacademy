Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the foundational language of the web. In this section, you’ll discover how HTML provides the basic structure for every website you visit. You’ll learn about essential elements such as headings, paragraphs, links, images, and lists, and how they work together to create meaningful web pages. We’ll guide you through the anatomy of an HTML document, explain the purpose of tags and attributes, and show you how to write clean, well-structured code. By the end of this lesson, you’ll be able to build simple web pages and understand how browsers interpret your markup to display content to users. This is the first step on your journey to becoming a web developer!

HTML Code
<!DOCTYPE html> <html> <head> <title>My First Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first web page.</p> </body> </html>

Browser Output

Hello, World!

This is my first web page.

HTML Document Structure

Every HTML document has a basic structure that includes:

  • <!DOCTYPE html> - Defines the document type
  • <html> - The root element
  • <head> - Contains meta information
  • <title> - Specifies the page title
  • <body> - Contains the visible content

Visual: HTML Document Structure

This diagram shows the basic structure of an HTML document.

Semantic HTML

Semantic tags like <header>, <nav>, <main>, <section>, <article>, <aside>, and <footer> describe the meaning of content, improving accessibility and SEO.

HTML Code
<header>Site Header</header> <nav>Navigation Links</nav> <main> <section>Main Content</section> <aside>Sidebar</aside> </main> <footer>Footer Info</footer>

HTML Attributes

Attributes add extra information to elements. Common attributes include id, class, style, title, and custom data-* attributes.

HTML Code
<p id="intro" class="highlight" title="Intro Paragraph">Hello!</p> <div data-user="student">Custom Data Attribute</div>

Comments in HTML

Comments are not displayed in the browser. Use them to explain your code or leave notes for yourself and others.

HTML Code
<!-- This is a comment and will not be shown --> <p>Visible content</p>

Special Characters & Entities

Use HTML entities to display reserved characters like <, >, &, and non-breaking spaces.

HTML Code
<p>Use &lt; and &gt; for < and > symbols.</p> <p>Non-breaking space: word&nbsp;word</p>

Links and Navigation

Create links with <a>. Use href for the destination. Anchor links (#section) jump to parts of the same page.

HTML Code
<a href="https://www.example.com">External Link</a> <a href="#top">Back to Top</a>

Real-World Example: Navigation Bar

The <nav> tag is used to group navigation links on a website.

HTML Code
<nav> <a href="index.html">Home</a> <a href="about.html">About</a> <a href="contact.html">Contact</a> </nav>

Images

Use the <img> tag to embed images. Always include the alt attribute for accessibility.

HTML Code
<img src="logo.png" alt="Logo">

Lists

Use <ul> for unordered lists, <ol> for ordered lists, and <dl> for description lists.

HTML Code
<ul><li>Apple</li><li>Banana</li></ul> <ol><li>First</li><li>Second</li></ol> <dl><dt>HTML</dt><dd>Markup language</dd></dl>

Tables

Tables organize data in rows and columns using <table>, <tr>, <th>, and <td>.

HTML Code
<table> <tr><th>Name</th><th>Age</th></tr> <tr><td>Alice</td><td>24</td></tr> </table>

Forms and Input Elements

Forms collect user input. Use <form> with <input>, <textarea>, <button>, <select>, and <option>.

HTML Code
<form> <input type="text" placeholder="Your name"> <input type="email" placeholder="Your email"> <button type="submit">Send</button> </form>

Media Elements

Embed images, audio, and video using <img>, <audio>, and <video>.

HTML Code
<img src="logo.png" alt="Logo"> <audio controls><source src="sound.mp3"></audio> <video width="320" height="240" controls> <source src="movie.mp4"> </video>

Document Metadata

The <head> section contains metadata like <title>, <meta>, <link>, and <script>.

HTML Code
<head> <title>My Page</title> <meta charset="UTF-8"> <link rel="stylesheet" href="style.css"> <script src="app.js"></script> </head>

Accessibility Basics

Use alt attributes for images, label for form fields, and semantic tags to make your site accessible to everyone.

HTML Code
<img src="cat.jpg" alt="A cute cat"> <label for="email">Email:</label> <input type="email" id="email">

Best Practices

Write clean, indented, and well-nested HTML. Use semantic tags and meaningful id/class names.

HTML Code
<!-- Good Example --> <section id="about"> <h2>About Us</h2> <p>Welcome to our site!</p> </section>

Links and Images

Creating Links

Use the <a> tag to create hyperlinks. The href attribute specifies the destination URL.

HTML Code
<!-- External link --> <a href="https://www.example.com">Visit Example</a> <!-- Internal link --> <a href="about.html">About Us</a> <!-- Link to email --> <a href="mailto:contact@example.com">Email Us</a>

Browser Output

Visit Example

About Us

Email Us

Adding Images

Use the <img> tag to embed images. The src attribute specifies the image path, and alt provides alternative text.

HTML Code
<!-- Local image --> <img src="logo.png" alt="Company Logo"> <!-- External image --> <img src="https://picsum.photos/200" alt="Random image"> <!-- Image with dimensions --> <img src="photo.jpg" alt="Landscape" width="400" height="300">

Browser Output

Company Logo

Lists and Tables

Creating Lists

HTML supports ordered (<ol>), unordered (<ul>), and description lists (<dl>).

HTML Code
<!-- Unordered list --> <ul> <li>Item 1</li> <li>Item 2</li> </ul> <!-- Ordered list --> <ol> <li>First step</li> <li>Second step</li> </ol>

Browser Output

Unordered List

  • Item 1
  • Item 2

Ordered List

  1. First step
  2. Second step

Creating Tables

Tables are created with the <table> tag. Use <tr> for rows, <th> for headers, and <td> for data cells.

HTML Code
<table> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>25</td> </tr> <tr> <td>Sarah</td> <td>30</td> </tr> </table>

Browser Output

Name Age
John 25
Sarah 30

Common Mistakes

  • Forgetting to close tags: <p>Hello (should be <p>Hello</p>)
  • Using inline styles everywhere instead of CSS classes
  • Missing alt text on images
Tag Purpose Example
<h1> Main heading <h1>Title</h1>
<p> Paragraph <p>Text</p>
<a> Link <a href="#">Link</a>

HTML & CSS Note

Remember: HTML is used to structure your web page, while CSS is used to style and visually enhance it. Keep your structure (HTML) and presentation (CSS) separate for cleaner, more maintainable code!

Form Elements

HTML Code
