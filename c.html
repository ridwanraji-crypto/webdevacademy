The Box Model

The CSS box model is the foundation of layout on the Web. Each element is represented as a rectangular box, with the box's content, padding, border, and margin built up around one another.

Box Sizing

The box-sizing property defines how the width and height of an element are calculated.

content-box (default)

Width = content width

Total width = width + padding + border

Width: 200px + 40px padding + 10px border = 250px
border-box

Width = content + padding + border

Total width = specified width

Width: 200px (includes padding and border)

Selector Specificity

Specificity determines which CSS rule is applied when multiple rules could apply to the same element.

  • Inline styles: 1000 points
  • ID selectors: 100 points
  • Class selectors: 10 points
  • Element selectors: 1 point

#nav .item:hover { }

Specificity: 100 (ID) + 10 (class) + 10 (pseudo-class) = 120

Transitions & Animations

CSS transitions and animations bring interfaces to life with motion.

Transitions

Transitions enable you to define the transition between two states of an element.

Hover Over Me

transition-property: Specifies the CSS properties to apply the transition to

transition-duration: How long the transition takes

transition-timing-function: The speed curve of the transition

transition-delay: When the transition will start

Shorthand:

transition: all 0.3s ease 0.1s;

Which is equivalent to:

transition-property: all;
transition-duration: 0.3s;
transition-timing-function: ease;
transition-delay: 0.1s;

Animations

CSS animations allow you to create complex animated sequences using keyframes.

Keyframes example:

@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.1); background: #f39c12; }
  100% { transform: scale(1); }
}

.element {
  animation: pulse 2s infinite;
}

Responsive Design

Responsive design ensures web applications work well on a variety of devices and window sizes.

Media Queries

Media queries allow you to apply CSS styles depending on device characteristics.

/* Mobile-first approach */
.container {
  width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    width: 750px;
  }
}

/* Desktop */
@media (min-width: 992px) {
  .container {
    width: 970px;
  }
}

Common Breakpoints:

  • Mobile: < 768px
  • Tablet: 768px - 991px
  • Desktop: 992px - 1199px

Calculating Specificity

1 Class (.item) = 10

1 Pseudo-class (:hover) = 10
